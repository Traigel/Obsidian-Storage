# unshift(...) shift()
### arr.unshift(...)
Добавить элемент в начало массива
```js
let arr = [1,2,3,4,5];
arr.unshift(0) // [0,1,2,3,4,5]
```

### arr.shift()
Уберает элемент из начала массива
```js
let arr = [1,2,3,4,5];
arr.shift() // [2,3,4,5]
```
