# push(...) pop()

### arr.push(...)
Добавить элемент в конец массива
```js
let arr = [1,2,3,4,5];
arr.push(6) // [1,2,3,4,5,6]
```

### arr.pop()
Уберает элемент из конца массива
```js
let arr = [1,2,3,4,5];
arr.pop() // [1,2,3,4]
```
